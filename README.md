# Solver - the best word game Solver

Solver is designed to help solve your favourite word games.  With a simple UI it gives access to multiple different tools for the same word pattern:
- Crossword word search (includes searching for phrases)
- Scrabble word search
- Anagram solver
- Anagram generator
- Word definitions
- Scrabble word checker
- Thesaurus

Solver comes with everything bundled so it works even when you are offline

## Layout

Solver always shows the current word pattern at the top of the screen, and you can use the tabs at the bottom to show different tools for that pattern.

The word pattern is a string where:
- letters are used for what you know
- question marks, dots and spaces represent single unknown letters
- plus signs or asterisks represent a sequence of unknown letters
- hyphens represent a break between words

## Word lists

Solver comes with multiple different word lists built in.  Use the app preferences to select the defaults for crosswords and scrabble tools separately.
